## 2D Heat Diffusion Solver (MATLAB)

## 📌 Problem
Simulation of heat diffusion in a 2D plate.

## ⚙️ Method
- Finite Difference Method (FDM)
- Explicit time stepping

## 🧮 Equation
∂T/∂t = α(∂²T/∂x² + ∂²T/∂y²)

## 🔧 Boundary Conditions
- Left wall: 100°C
- Right wall: 25°C
- Top & Bottom: Insulated

## ▶️ How to Run
Run `main.m`
